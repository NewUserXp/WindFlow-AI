# Release
### We're very excited to announce WindFlow today!

### What is WindFlow?
WindFlow AI is an AI platform where you can try out **Claude 3** AI model for free. We're offering **250 requests/day** of rate limits.
To access our API use: api.betawindflow.net/v1 and type as model: "anthropic/claude-3" to access the API.

### Allowed API Parameters:
- ✅**Stream**: Stream the response of the AI **live**. Parameter: ```stream: true```
- ✅**Enable Thinking**: Toggle thinking of the AI. Parameter: ```enable_thinking: true```
- ❌**Tool Calling**: Tool calling is currently not supported, we're working on it (Will release in 1-2 weeks!) Parameter: ```tools:```
- The rest of the OpenAI compatible ones.

### API Key Creation?
Go to our website (legacy.windflow.ai) and go to API Keys. Click **Create API Key** to create an API Key.
- **NOTE:** The API key is stored on your IP address to prevent abuse. We're working on an account system and are about to release it in 2025 January, February or March!

## Exact Rate Limits:
- 5 requests per minute
- 40k tokens per minute
- 250 requests per day
- 750k tokens per day
**NOTE:** These rate limits may change at anytime.

**Rate Limiting is required to prevent abuse.**
